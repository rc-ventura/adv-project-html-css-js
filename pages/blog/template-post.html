Dra. Laura Bender 10 de Novembro, 2023

Este é o parágrafo de introdução. Ele geralmente tem um destaque sutil para prender a atenção do leitor logo no início do texto.

Aqui começa o texto do seu artigo. Você pode escrever parágrafos normais usando a tag <p>. A tipografia e os espaçamentos já estão configurados para facilitar a leitura no computador e no celular.

Subtítulos devem usar H2

Sempre que for mudar de tópico dentro do artigo, use um <h2>. Isso ajuda muito o Google a entender a estrutura do seu texto.

  • Você pode fazer listas de itens
  • Como esta aqui
  • Para facilitar a leitura dinâmica

Caso queira dar destaque a uma citação ou lei, você pode usar um bloco de citação:

"A justiça é o vínculo das sociedades humanas; as leis emanadas da justiça são a alma de um povo."

Um tópico menor (H3)

Se precisar de um subtópico dentro do H2, use o H3. Lembre-se de sempre manter os textos em blocos curtos para a leitura não ficar cansativa na tela do celular.

Para adicionar imagens no meio do texto, basta usar a tag de imagem normal. O CSS fará com que ela se ajuste automaticamente na tela.
